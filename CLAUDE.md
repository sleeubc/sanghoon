This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## Package Overview

`sanghoon` is a personal R utility package. All functions live in `R/` as individual files, each documented with roxygen2.

## Common Commands

```r
# Load package interactively during development
devtools::load_all()

# Rebuild documentation from roxygen2 comments
devtools::document()

# Run checks
devtools::check()

# Install locally
devtools::install()
```

## Architecture

Each function in `R/` is self-contained and uses `require()` internally to load its dependencies. The NAMESPACE is generated by roxygen2 — do not edit it manually.

### Function Summary

| File | Purpose |
|------|---------|
| `classify.R` | Discretize numeric vector into interval factor (equal/quantile/jenks breaks) |
| `delineate_watershed_cities.R` | Watershed delineation from smoothed raster using whitebox tools |
| `duplicates_by.R` | Return all duplicate rows (including first occurrence) by specified columns |
| `left_join_m.R` | `dplyr::left_join` wrapper that prints match type (1:1, m:1, etc.) and unmatched row count |
| `lm_cluster.R` | `lm()` wrapper that attaches clustered SEs (`$se_cluster`, `$vcov_cluster`) for use with `stargazer` |
| `se_mean.R` | Standard error of the mean with `na.rm` support |
| `summarise_combo.R` | Run `dplyr::summarise` for all combinations of `group_by` variables |
| `unlist_null2na.R` | Convert NULL elements to NA before `unlist()` |

## Adding a New Function

1. Create `R/<function_name>.R` with roxygen2 `@export` tag.
2. Run `devtools::document()` to update NAMESPACE and `man/`.
3. Commit both the `R/` file and generated `man/<function_name>.Rd`.
